# Skylanders Wiki

Hybrid mobile app, Skylanders data Wiki.

## Technologies :

- Typescript / React Native
- Expo
- fetch for API calls
- React Navigation

## Wiki

This app uses this [API](https://github.com/MatteoBernard/SkylandersApi) .
You can access to various datas, related to the Skylanders franchise.
